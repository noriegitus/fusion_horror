# Fusión Horror
### Descripción:
Generación de historias de terror en base a palabras clave del usuario.
El usuario puede especificar hasta 3 features para el generador.

### Lenguajes y Bibliotecas
- Python 
- Pytorch
- Transformers
- GPT-2 Open Source Model
- Tensorflow
- Pandas
- Matplotlib, Plotly
- Streamlit
